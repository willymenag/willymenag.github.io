Custom Domain
Use your own domain like willymena.com instead of willymenag.github.io.
How to set up a custom domain:

1. Buy a domain from Namecheap, GoDaddy, or similar (~$10/year)
2. In your domain's DNS settings, add these A records pointing to GitHub Pages:
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
3. Add a CNAME record for www pointing to willymenag.github.io
4. Enter your custom domain below and click Save — it will create a CNAME file in your repo
5. Go to your GitHub repo → Settings → Pages → set your custom domain there too
6. Wait 24 hours for DNS to propagate — done! 🎉
